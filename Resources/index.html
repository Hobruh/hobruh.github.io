Color Syntax in CSS

Color Tags

Tag Name Description
Background-color Background Color Changes the background of an element
Color Foreground Color Changes the color of whatever is inside of an element

Color Types

Property Name Description
Color CSS Colors The base 140 colors CSS can render
HSL Hue Saturation Color Defines color based on the varying levels of hue, saturation, and color.
RGB Red Green Blue Defines color based on the level of red, green, and blue coloring.
#000000 HexDecimal Defines color based on alphanumerical values associated with various colors.
A Opacity Added to the end of HSL, RGB, or HexDecimal property to define the level of transparency a color has
